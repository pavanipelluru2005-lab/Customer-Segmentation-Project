# Customer-Segmentation-Project
A machine learning project that segments customers into different groups based on annual income and spending score using K-Means Clustering. The project helps identify customer behavior patterns and supports targeted marketing strategies.
## Description

This project uses the K-Means Clustering algorithm to segment customers based on their annual income and spending behavior. By grouping similar customers into clusters, businesses can better understand customer preferences, identify high-value customers, and create targeted marketing campaigns.

### Features
- Customer Segmentation using K-Means Clustering
- Customer Behavior Analysis
- Data Visualization with Scatter Plot
- Identification of High-Value and Low-Value Customers
- Business Insights for Targeted Marketing

### Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab
